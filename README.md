# Dokumentasi FrondEnd

## Persyaratan

- Node.js v20 atau lebih baru
- NPM v10 atau lebih baru

## Instalasi

1. Masuk ke direktori proyek:
   ```bash
   cd kamifront
   ```
2. Instal dependensi:
   ```bash
   npm install
   ```

## Menjalankan Server

Untuk menjalankan FE, gunakan perintah berikut:

```bash
npm run dev
```

Server akan berjalan di `http://localhost:5137`.
